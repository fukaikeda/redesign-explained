Redesign of Kathy's Driving School

While searching for a driving school, I found it difficult to read websites and find a service that appealed to me. Applying my knowledge of user interfaces, I identified key usability issues that hindered the user experience. Therefore, I decided to redesign a webpage to improve accessibility and navigation, ensuring a seamless experience across different devices.

The Original Website

The original website I chose for this redesign is Kathy's Driving School. When attempting to book a session, I noticed several usability issues, including unnecessary animations and unclear navigation.

Accessibility and Usability Issues

Through my analysis, I identified the following key issues with the website:

  • Efficiency: Booking a session is difficult because users must select a location before viewing the calendar. This is frustrating for users who prioritize availability over location.
  • Comparing Prices: Users cannot compare prices easily, which may be intentional but negatively impacts decision-making.
  • Learnability: Service descriptions are unclear; the site immediately directs users to pricing and booking pages instead of explaining the offerings.
  • Payment Procedure: The payment process lacks transparency, making it hard for users to understand how to complete transactions.
  • Memorability: The site contains multiple links leading to different calendars with inconsistent UI, making navigation confusing.
  • Conceptual Model: The site's icons and banners do not correspond to their content, causing disorientation.

Web Accessibility Audit

Using WebAIM WAVE, I evaluated the accessibility of the website. Key findings include:

  • Alt text is provided for images, which is good practice. However, the descriptions are not informative enough for screen readers.
  • Some text elements have low contrast, making readability difficult for users with visual impairments.
  • Several form inputs are missing labels, making it challenging for screen reader users to interact with them.

To improve accessibility, I recommend adding detailed alt text, enhancing text contrast, and ensuring all form inputs have clear labels. View my annotations here.

Redesign Goals

My primary goal is to make the website more user-friendly, ensuring users can:

  • Swiftly navigate and find relevant information.
  • Easily compare pricing and services.
  • Understand the booking and payment processes without confusion.
  • Experience an accessible and visually clear interface.

Style Guide

To ensure a cohesive and accessible design, I developed a style guide focusing on readability, consistency, and user experience. The redesign adopts a clean and modern aesthetic based on the original theme.

The primary color palette consists of high-contrast colors based off the Rhode Island licsense plate to enhance readability, particularly for users with visual impairments. I selected an Arial typeface for its legibility.

Elements, such as buttons and graphs, follow a consistent design pattern to improve usability. Additionally, I implemented a responsive layout, allowing seamless adaptability across different screen sizes.

This style guide ensures that the redesigned website maintains both visual appeal and accessibility, aligning with modern web design best practices.

Conclusion

Through this project, I gained valuable insights into usability issues and accessibility challenges. By implementing a structured redesign, I aimed to enhance user experience, making the website more efficient, learnable, and inclusive for all users.

Key improvements to the website include removal of unnecessary dynamic components, restructure of important functionality like booking sessions, and an overall more consistent design. By implementing a modern color palette and a basic font, the website has a cleaner look as well as a higher legibility. This experience furthered my appreciation for a good design and developed my practical skills in improving accessibility.
